# Spam Message Detector

## Description
A simple machine learning project to classify text messages as Spam or Not Spam.  
Built using Python and scikit-learn with CountVectorizer and Logistic Regression.

## Features
- Converts text messages into numeric vectors using CountVectorizer  
- Trains a Logistic Regression model for classification  
- Predicts whether a new message is spam or not  
- Beginner-friendly ML project

## How to Run
1. Clone the repository  
2. Install dependencies:
```bash
pip install -r requirements.txt
python spam_detector.py
Input: "You have won cash prize"
